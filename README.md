# 🎬 CineVau

CineVau is a movie discovery website built for people who love exploring films and TV shows. It features a clean, modern design and a smooth user experience for browsing entertainment content.

🌐 **Live Demo:** https://cinevau.netlify.app/

## Features

* Modern and responsive UI
* Browse movies and entertainment content
* Fast and user-friendly experience
* Works across desktop and mobile devices

## Tech Stack

* HTML
* CSS
* JavaScript
